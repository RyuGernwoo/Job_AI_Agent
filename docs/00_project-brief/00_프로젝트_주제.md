# [가천 AI부트캠프사업단] 26-여름 현장미러형 프로젝트

## [프로젝트 4번] 직업훈련 강의 운영 보조 AI AGENT

### 교안·실습·평가 자동 생성

## 해결하고 싶은 문제

- 생성형 AI 교육 사업에서 과정별 교안, 실습 과제, 평가 문항을 매 차수 수작업으로 제작하여 강사 업무 부담이 큼
- NCS 능력단위와 커리큘럼을 근거로 일관된 산출물을 생성해야 하나 표준화와 근거 추적이 어려움
- 강사마다 산출물 형식·난이도가 달라 품질 편차가 발생함

## 기대하는 결과물

- 커리큘럼·NCS 능력단위 입력 시 차시별 교안 초안, 실습 시나리오, 객관식·실습형 평가 문항을 생성하는 Agent 구현
- 보유 교재를 근거로 인용하는 RAG 연동으로 할루시네이션 억제
- 강사 검수·수정이 가능한 휴먼인더루프(Human-in-the-loop) 워크플로우 구성
- FastAPI API와 산출물 다운로드(DOCX·PPTX) 기능 제공

## 관련 기술 분야

- AI Agent
- RAG
- CLOVA Studio·HyperCLOVA X
- FastAPI
- 휴먼인더루프(HITL)

## 학생들이 참고할 사항

- 샘플 커리큘럼·교재와 출력 문서 형식 가이드를 제공함
- 저작권 및 근거 출처 표기 준수를 핵심 요건으로 설정함
- 생성 산출물의 일관성·재현성 평가 기준을 함께 정의함
